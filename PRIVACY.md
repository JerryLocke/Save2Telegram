# Save2Telegram Privacy Policy

Save2Telegram is a Chrome extension that helps users forward media from X/Twitter posts to a Telegram channel configured by the user.

## Data the extension uses

The extension may store the following data locally in the user's browser:

- Telegram Bot Token
- Telegram channel ID or channel username
- User-created Telegram forwarding configurations
- Forwarding queue records and task status
- Extension preferences
- Optional self-hosted backend endpoint URL and endpoint key

When the user actively forwards a post, the extension may process the selected X/Twitter post URL, post text, author display information, image URLs, video URLs, and media files for the purpose of sending that content to the user's configured Telegram channel.

## How data is used

The data is used only to provide the extension's single purpose: forwarding user-selected X/Twitter media to a Telegram channel.

The extension does not sell user data. The extension does not use user data for advertising, credit evaluation, lending, or unrelated analytics.

## Data sharing

When the user starts a forwarding action, the selected content and Telegram configuration are sent to Telegram's Bot API so the media can be delivered to the user's selected Telegram channel.

If the user chooses to bind a self-hosted forwarding backend, forwarding data may be sent to that backend to process media download and upload tasks. This backend is configured and controlled by the user.

## Remote code

The extension does not load or execute remote code. JavaScript, CSS, and HTML used by the extension are included in the extension package.
